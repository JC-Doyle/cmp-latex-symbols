# cmp-latex-symbols

Add latex symbol support for [nvim-cmp](https://github.com/hrsh7th/nvim-cmp).

```lua
use({
  "hrsh7th/nvim-cmp",
  requires = {
    { "kdheepak/cmp-latex-symbols" },
  },
  sources = {
    { name = "latex_symbols" },
  },
})
```

Based on [compe-latex-symbols](https://github.com/GoldsteinE/compe-latex-symbols/).
